➕ Nieuw filiaal toevoegen
💡 De Sleutel is de interne code (bijv. HASSELT). Gebruik enkel hoofdletters, geen spaties — gebruik koppelteken voor meerdere woorden. Postcodes en gemeenten koppel je daarna via Toewijzing beheren.

Elke regel = één tijdslot. Gebruik Enter voor een nieuwe regel.

Ben je zeker dat je wil verwijderen?
Dit verwijdert het filiaal uit filialen_data.json. Koppelingen in city_lookup.json moeten apart worden verwijderd via Toewijzing beheren.
